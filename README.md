# Jogo da Velha 🎮

Um jogo da velha (Tic-Tac-Toe) desenvolvido com HTML, CSS e JavaScript puro, com suporte a dois jogadores e detecção automática de vitória e empate.

## 🖥️ Funcionalidades

- Cadastro dos nomes dos dois jogadores antes de iniciar
- Indicação do jogador da vez atualizada a cada turno
- Tabuleiro 3x3 interativo com marcação de X e O
- Detecção automática de vitória com destaque visual nas células vencedoras
- Detecção de empate quando o tabuleiro está completo
- Botão para reiniciar a partida a qualquer momento

## 🛠️ Tecnologias

- HTML5
- CSS3
- JavaScript (Vanilla)
